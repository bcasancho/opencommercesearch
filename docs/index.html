API Endpoints

The OpenCommerceSearch API provides methods for accessing resource like brands, products, categories, etc a canonical URL. For example https://api.backcountry.com/v1/brands/88

In some cases, you can perform actions like search or suggest completion. For example https://api.backcountry.com/v1/brands/suggestcompletion/north

These is the list of available resources:

  • brands
  • categories
  • products
